# YOLOv8 Real-Time Object Detection with Webcam

## 📌 Project Overview
This project demonstrates **real-time object detection** using a webcam and the **YOLOv8 Nano model**.  
It captures live video feed, detects objects instantly, and displays bounding boxes with confidence scores and class labels.

The project is built using **Python**, **OpenCV**, **Ultralytics YOLOv8**, and **CvZone**.

---

## 🚀 Features
- Real-time object detection using webcam
- Uses pretrained **YOLOv8n** model
- Displays:
  - Bounding boxes
  - Confidence scores
  - Object class names
- Mirror effect for better user interaction
- Stylish UI with `cvzone.cornerRect()`
- Press **Q** to quit anytime

### 🧠 How It Works
- Opens the webcam feed

- Loads the pretrained YOLOv8 Nano model

- Detects objects frame by frame

- Draws bounding boxes with confidence scores

- Displays detected object names in real time
