# Dataset Download Guide

This guide explains how to download the dataset from Kaggle step by step.

---

# Step 1: Open the Kaggle Dataset Page

Go to the dataset page in Kaggle using your browser.<br>

Click on Data Hub that is on your left side. <br>

Then,click on Datasets as shown on the step1.png. <br>

![Step 1](https://github.com/sandesharyal-mrcreative369/YOLO-Projects/blob/main/car-counter-project/Datasets/step1.png)

---

# Step 2: Sign In to Kaggle

If you are not logged in, click on **Sign In** and log into your Kaggle account.<br>

After sign-in the kaggle serach Car Detection and Tracking Dataset on search bar. <br>

Or If you can't find out then copy and paste the link below:-
- https://www.kaggle.com/datasets/amitkumargurjar/car-detection-and-tracking-dataset

![Step 2](https://github.com/sandesharyal-mrcreative369/YOLO-Projects/blob/main/car-counter-project/Datasets/step2.png)

---

# Step 3: Open the Download Section

After opening the dataset page, locate the **Download** button on the right side. <br>

After locating the **Download** button. <br>

- Click on the **Download dataset as zip** button to start downloading the dataset ZIP file. <br>

![Step 3](ihttps://github.com/sandesharyal-mrcreative369/YOLO-Projects/blob/main/car-counter-project/Datasets/step3.png)

---


# Step 5: Extract the ZIP File

After downloading is completed:

- Locate the ZIP file
- Right-click the file
- Select **Extract Here** or **Extract All**


---

# Step 6: Move Dataset to Project Folder

Copy or move the extracted dataset folder into your project directory.

- Note: you have to create a <b>outputs</b> folder in your repository and stored the extracted zip file.

Example:

```text

project/
│
|outputs/
│   └── archive/
│       └── train/
│           └── images/
│               ├── 1.jpg

│               ├── ...
|
|
├── Dataset/
│   ├── DataSet.md
│   ├── step1.png
│   ├── step2.png
